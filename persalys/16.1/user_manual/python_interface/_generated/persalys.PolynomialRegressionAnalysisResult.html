PolynomialRegressionAnalysisResult¶

class persalys.PolynomialRegressionAnalysisResult(*args)¶

Get the results of a Linear model analysis.

Notes

Structure created by the method run() of PolynomialRegressionAnalysis, and obtained thanks to the method getResult().
